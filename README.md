## Features

- CSS Module
- Sanctum

## Getting started

```bash
# 1. Clone the repository.
git clone https://github.com/Aslam97/laravel-vue-boilerplate.git my-new-lavueject

# 2. Enter your newly-cloned folder.
cd my-new-lavueject

# 3. Install composer dependencies
composer install

# 4. Install npm dependencies
npm install
```
